# trabalhando-com-python
trabalhando com banco de dados em python
No momento, até que estou gostando de Python. Era para ser uma linguagem de programação 
fácil, porém há algumas dificuldades para mim, principalmente quando acho que o código está 
certo e, quando vou testar, dá erro — é frustrante.
Porém, entendo que estou aprendendo e ainda tenho muito mais para aprender. O que mais me 
chamou a atenção e que eu mais gostei foi a interação com o sistema de poder interagir com o 
usuário.
O professor nos ajuda muito, faz bastante exercícios e dinâmicas.
Minhas expectativas para as próximas aulas são ganhar confiança para fazer códigos sem 
precisar consultar o que já fizemos, além de poder criar mais projetos e montar um 
portfólio, pra assim conseguir alguma oportunidade de emprego.
